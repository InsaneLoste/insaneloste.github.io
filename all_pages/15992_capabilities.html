The user has access to an object that grants them monster-based powers. Whether they are accessories, tools or weapons, the objects are capable of enabling powerful abilities associated with their particular monster power. They can be supernaturally imbued can be possessed by some type of entity associated with monsters. The user can bond with the entity (Monster Symbiosis) or wield it as a living tool. Powerful as they can be, the user must be careful not to be overwhelmed by the force/energy wielded.
