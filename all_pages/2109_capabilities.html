The user can interact with things normally with great pulling capabilities. Basically, this works like this: The user can pull or lift objects from the floor, ceiling, walls or inside another object without needing any doorknobs, handles, ridges, handholds and footholds, etc. The user can also ride objects and stick to them even if they do not have straps, seat belts or the like. This can also be done with attractive forces such as gravity and magnetism.
