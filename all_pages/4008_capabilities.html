The user can act, attack and move any sudden changes done to a solid or liquid surface by anything. This can be used to defy changes such as changes in the shape and size of the object caused by anything such as explosives, earthquakes, impacts, superpowers, technology, etc. Unlike Moving Surface Activity, which deals with the motion of surfaces, this power deals with changes in shape and size of the surface only. This can be done no matter how fast a surface changes.
