• Fixed points in time can be used in both positive and negative ways, such as if a situation that would be unwanted is temporally anchored, there would be nearly no way to cancel it.
  • Though the user can't be killed, they can still be trapped.
  • Absolute Destruction, Absolute Erasure, Absolute Death Inducement, Meta Event Manipulation, Meta Time Manipulation, Concept Destruction can end points in time despite them being "fixed."
  • Weak against powers that deal with possibilities and time such as Variable Collapse, Time Destruction/Temporal Erasure, Temporal Bypassing, etc.
  • Depending on the circumstances one became a fixed point in time, Absolute Restoration may be able to undo the lock.
  • Time Manipulators, Space-Time Manipulators are able to actually undo this and remove this fixed point in space-time and and undo the events that were set in stone thus allowing them to be changed/altered again.
